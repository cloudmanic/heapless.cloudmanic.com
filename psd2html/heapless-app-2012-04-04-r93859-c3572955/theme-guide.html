General Notes

Because we are forward thinkers looking to the future, we target the latest browsers including:

  • Latest Google Chrome
  • Firefox 4+
  • Opera 11
  • Latest Safari
  • Internet Explorer 9+

All modal windows should be built using http://jacklmoore.com/colorbox (colorbox). This is our preferred solution.
